Project Details:

Mission

To make a simple Java-based tool that will enable users to easily manage their tasks.

Vision

An easy to use tool that allows company staff to visualise, schedule and prioritise their tasks. It is highly extensible so that future iterations can be utilised by clients and integrated with other projects.

Project Size

Small

Scope (Minimal Viable Product)

A bespoke to-do list application that follows best practices to meet the following functional requirements:

         R1 The system is able to display a list of tasks to be completed

         R2 The user is able to add a new task

         R3 The user is able to edit an existing task

         R4 The user is able to delete an existing task

         R5 The user is able to schedule a task for completion on a particular date

         R6 The system is able to display an overview of the tasks to be completed on a weekly calendar in accordance with their scheduled completion date

         R7 The system is able to sort the list of tasks in descending order based on scheduled completion date.

         R8 The system is able to display a detailed view of a particular task including the task name and scheduled completion date

 

The application must also meet the following non-functional requirements:

         NFR1 The application should store all data in an efficient and lightweight database

 

         NFR2 The application must be platform independent so that it can be used on desktop, mobile and web.

 

         NFR3 The application must have a simple and efficient user interface so that the user is able to complete any function within 5 seconds when starting from the home screen.

 

         NFR4 The application must utilise a supported version of the Java Development Kit (JDK) minimum JDK 8.

Success Criteria

MySoftwareDev Co. consider this project to be successful if:

         The user is able to navigate through the application to complete any given user story within 5 seconds when starting from the home screen.

         The application takes up less than 256mb of storage space when installed on a desktop device.

         Users are able to view a list of their upcoming tasks.

         Users are able to add, view, edit, and delete tasks from the to-do list.

         Users are able to see when their tasks are due for completion based on the scheduled completion date.

         The interface is suitable for desktop, mobile and web platforms with a consistent look and feel regardless of the device type.
