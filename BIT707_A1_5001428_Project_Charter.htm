Project Details:

Mission

To make a simple Java-based tool that will enable users to easily manage their tasks.

Vision

An easy to use tool that allows company staff to visualise, schedule and prioratise their tasks. It is highly extensible so that future iterations can be utilised by clients and integrated with other projects.

Project Size

Small

Scope (Minimal Viable Product)

A bespoke to-do list application that follows best practices to meet the following requirements:

·         The ability for users to add and delete tasks from a list

·         The ability to schedule a task to be completed on a particular date

·         A user view (calendar-view) that dispalys an overview of tasks on a weekly calendar based on their scheduled completion date

·         A user view (list-view) displaying an overview of tasks in the list, sorted in descending order based on scheduled completion date.

·         The ability to add or delete tasks from the list-view

·         A user view (task-view) that displays the details of a particular task and allows users to update the task, delete it, or add a new task

 

The application must be equipped with a lightweight database to efficiently store data on the users device.

 

It must have a clean interface and be platform independent so that it can be used on desktop, mobile and web.

 

It must utilise a supported version of the Java Development Kit (JDK)  minimum JDK 8.

Success Criteria

MySoftwareDev Co. consider this project to be successful if:

·         The application is easy to use and can be utilised by their staff or customers with little or no training

·         The application is efficient and includes a lightweight database to minimise storage space requirements.

·         The interface is simple and quick to use.

·         The application makes it easy for users to manage their tasks and see when they are due for completion

·         The interface is suitable for desktop, mobile and web platforms.
